Anchors

^Start of string (or line with m flag)
$End of string (or line with m flag)
\bWord boundary
\BNon-word boundary
\AStart of string (no multiline)
\ZEnd of string (no multiline)

Character Classes

.Any character except newline
\dAny digit [0-9]
\DNon-digit
\wWord char [a-zA-Z0-9_]
\WNon-word character
\sWhitespace [ \t\r\n\f]
\SNon-whitespace
[abc]Any of: a, b, c
[^abc]Not any of: a, b, c
[a-z]Character range a through z

Quantifiers

*0 or more (greedy)
